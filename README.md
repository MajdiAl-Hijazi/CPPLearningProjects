# Array Utilities in C++

This C++ project contains multiple functions for working with arrays of integers. It is designed for learning array manipulation and common algorithms.

## Features

- Fill an array with random numbers or sequential numbers.
- Print arrays of integers or strings.
- Swap two numbers in the array.
- Search for a number in an array.
- Shuffle arrays in different ways.
- Copy arrays entirely, in reversed order, or selectively (odd numbers, prime numbers, distinct elements).
- Generate random keys in the format `AAAA-BBBB-CCCC-DDDD`.
- Check if an array is a palindrome.
- Count odd numbers, even numbers, and positive numbers in an array.

## How to Use

1. Compile the code using a C++ compiler (e.g., g++, Visual Studio, Code::Blocks).
2. Run the executable.
3. Follow on-screen instructions for array size, filling options, and operations.

# أدوات التعامل مع المصفوفات في C++

يحتوي هذا المشروع بلغة C++ على عدة دوال للتعامل مع مصفوفات الأعداد الصحيحة. تم تصميمه لتعلم التعامل مع المصفوفات والخوارزميات الشائعة.

## المميزات

- تعبئة المصفوفة بأرقام عشوائية أو متسلسلة.
- طباعة مصفوفات الأعداد الصحيحة أو النصوص.
- تبادل رقمين داخل المصفوفة.
- البحث عن عنصر داخل المصفوفة.
- خلط عناصر المصفوفة بطرق مختلفة.
- نسخ المصفوفة كاملة، بالترتيب العكسي، أو بشكل انتقائي (الأرقام الفردية، الأرقام الأولية، الأرقام المميزة).
- توليد مفاتيح عشوائية بصيغة `AAAA-BBBB-CCCC-DDDD`.
- التحقق إذا كانت المصفوفة متناسقة (Palindrome).
- عد الأرقام الفردية، الزوجية، والأرقام الموجبة في المصفوفة.

## كيفية الاستخدام

1. ترجمة الكود باستخدام مترجم C++ (مثل g++, Visual Studio، أو Code::Blocks).  
2. تشغيل البرنامج التنفيذي.  
3. اتباع التعليمات على الشاشة لتحديد حجم المصفوفة، خيارات التعبئة، والعمليات المطلوبة.
